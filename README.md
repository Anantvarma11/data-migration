# Enterprise Migration Services Platform

A production-ready frontend dashboard for Enterprise Data Platform Migration Services.
This application uses React 18, TypeScript, Vite, and Material-UI.

## Features

- **Transactional DB Migration**: AI-Augmented Replatform Strategy visualization.
- **Data Warehouse Migration**: Autonomous Modernization Strategy with layered architecture.
- **Bulk File Data Migration**: Network-First strategy with integrity checks.
- **Lift-and-Shift**: Rapid infrastructure exit strategy visualization.
- **Responsive Design**: Mobile-friendly sidebar and layout.
- **Enterprise UI**: Google Cloud inspired theme using MUI v5.

## Project Structure

```
/src
 ├── components
 │   ├── common      # Layout, Navbar, Sidebar
 │   └── sections    # Dashboard & Strategy Pages
 ├── theme           # MUI Theme Configuration
 ├── assets          # Static assets
 ├── App.tsx         # Routing Logic
 └── main.tsx        # Entry point
```

## Setup & Installation

1. **Prerequisites**: Node.js (v18+ recommended) and npm.
2. **Install Dependencies**:

    ```bash
    npm install
    ```

## Running Locally

Start the development server:

```bash
npm run dev
```

Access the app at `http://localhost:5173`.

## Building for Production

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
